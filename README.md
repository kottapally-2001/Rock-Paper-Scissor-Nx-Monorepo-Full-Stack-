
# Rock Paper Scissors - Nx Monorepo (Refine + NestJS)

## How to use

1. Install dependencies:

```bash
npm install
```

2. Run the backend (NestJS):

```bash
npx nx serve backend
```

This starts the backend on http://localhost:3000

3. In another terminal, run the frontend (Refine + React):

```bash
npx nx serve frontend
```

This starts the frontend on http://localhost:5173

Open the frontend URL in your browser and play the game.
